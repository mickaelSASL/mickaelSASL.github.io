Indexation des données en tables¶

Consulter les données dans un fichier csv¶

Vous allez manipuler les données de deux tables de données contenant des informations concernant les monuments nationaux français.
Ces deux tables sont construites de la façon suivante : (source www.data.gouv.fr)

Fichier "monuments_frequentation.csv" :
id nom frequentation
id : numéro de l'enregistrement
nom : nom du monument
frequentation : nombre de visiteurs en 2012
ET
Fichier "monuments_GPS.csv" :
id nom lat long
id : numéro de l'enregistrement
nom : nom du monument
lat : latitude du monument
long : longitude du monument

Fichiers à télécharger

  • Clic droit --> Enregistrer le lien sous..
  • Enregistrer les 3 fichiers dans le même dossier
  • Créer un fichier Pyhton dans lequel vous recopierez les bouts de codes ci-dessous

monuments_frequentation.csv
monuments_GPS.csv
tables.py

Pour la suite du travail vous aurez besoin des bibiliothèques csv, tables.

Lire les données¶

d'un fichier .csv sous la forme d'un dictionnaire Python.
rappel : un dictionnaire est une structure de données présentée sous la forme : {clé_1: valeur_1 ; clé_2: valeur_2}

In [3]:
import csv
csv_file = open('monuments_GPS.csv', "r")
reader = csv.DictReader(csv_file)

for row in reader:
    print(dict(row))

csv_file.close()

Carte interactive

Créer une page html affichant une carte intéractive des monuments nationaux.
La page affichera une vue de la France métropolitaine avec un marqueur à l'emplacement de chaque monument.
Le marqueur sera de couleur différente suivant la fréquentation : 
+ de 100 000 visiteurs : Rouge,
- de 100 000 visiteurs : vert

Pour les plus avancés :

Compléter la carte en ajoutant :

1) Une bulle de survol sur chaque marqueur indiquant le nom du monument.
2) Une légende (au clic) qui rappellera le nom du monument et le nombre de visiteurs.
3) Une image du monument à la légende. (les images ne s'affichent pas sur l'aperçu dans la console)
